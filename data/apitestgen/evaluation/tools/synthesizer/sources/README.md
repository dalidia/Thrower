Synthesize stack traces (`.log` files) from a JSON file. 

# Usage

```
usage: java -jar stacktrace-synthesizer.jar <options>
 -c <case-name>         The name of the case as used by ExRunner (mandatory).
 -h                     Prints this message
 -i <input-file.json>   JSON file generated by eRec (mandatory).
 -o <output/>           The output directory where the stack traces and mapping file will be generated (mandatory).
```

## Example

```
java -jar stacktrace-synthesizer-0.0.1-SNAPSHOT-jar-with-dependencies.jar -c commons-lang3-3.7-simple -i commons-lang3-3.7-simple.json -o test-synthesizer3
```
